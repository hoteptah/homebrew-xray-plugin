# 🍺 xray-plugin Homebrew Tap
A custom Homebrew tap for installing **xray-plugin*, enabling Shadowsocks to use Xray-based transport features such as TLS and advanced obfuscation. Binaries are sourced from the official upstream releases: https://github.com/teddysun/xray-plugin

---

## 📦 Installation

### 1. Add the tap

```bash
brew tap hoteptah/xray-plugin
```

### 2. Install the plugin

```bash
brew install xray-plugin
```
### 3. Verify installation

```bash
xray-plugin --version
```

## ⚠️ Supported Platform

Currently, this tap installs the binary from the following upstream release:

https://github.com/teddysun/xray-plugin/releases/download/v1.8.24/xray-plugin-linux-amd64-v1.8.24.tar.gz

**Supported platform:**

OS: Linux
Architecture: amd64 (x86_64)

*Other architectures (ARM, etc.) are not supported yet.*
